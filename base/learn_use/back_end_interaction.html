后端服务项目 ✳️ ​

1.下载后端服务 ​

为了完美的实现前后端交互，我们需要有一个后端服务，这里我们使用node.js来搭建一个简单的后端服务,你只需下载下方的压缩包解压。

后端服务.zip

解压后进入项目目录，运行pnpm install安装依赖。

bash
pnpm install
1

2.启动后端服务 ​

运行pnpm run dev启动后端服务。

bash
pnpm run dev
1

当看到控制台打印出Server is running on port 3000字样时，说明服务已经启动成功,可以通过http://localhost:3000访问服务。

前端服务项目 ✳️ ​

1.开启vite代理服务配置 ​

变更项目根目录vite.config.ts的server配置：

typescript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import AutoImport from 'unplugin-auto-import/vite'
import Components from 'unplugin-vue-components/vite'
import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'
export default defineConfig({
  plugins: [vue(),
    AutoImport({
    resolvers: [ElementPlusResolver()],
  }),
    Components({
      resolvers: [ElementPlusResolver()],
    }),],
  server:{
    host: '0.0.0.0',//允许外部访问
    port: 8888, //前端服务开启的端口号
    open: true, //是否自动打开浏览器
    proxy:{     //代理配置
      '/api':{  //当接听到请求路径中以/api开头的请求时，将其转发
        target:"http://localhost:3000", //转发目标地址(后端服务地址)
        rewrite:path => path.replace(/^\/api/, '') //转发前将/api替换为空字符串
      }
    }
  }
})

说明

  • 为什么要配置server，因为我们前后端服务的端口不一样，在浏览器端有一种安全机制(CORS跨域资源共享)，如果不配置server，浏览器会拦截响应结果,导致前端无法接到后端服务的响应数据。
  • 这个配置项,相当于中间件，将请求转发到后端服务(可以简单理解为nginx类似的服务)。

2.基础封装axios ​

在src/api下新建index.ts文件，用于封装axios请求。

typescript
//导入axios
import axios from 'axios';
//创建axios实例
const api = axios.create({
    baseURL: '/api',//将全部请求的前缀都加上/api便于vite转发
    timeout: 5000,//超时时间
    headers: {
        'Content-Type': 'application/json',//默认请求头
    }
});
//定义响应数据类型
type responseType<T> = {
    code: number;
    data: T;
    message: string;
}
//请求响应拦截器
api.interceptors.response.use((response)=>{
    //对响应数据做处理
    //对响应数据做处理
    if(response.data && response.data.code === 200) {
        return response.data;
    }
},(error)=>{
    //请求错误时做一些处理
    console.log(error)
})
//统一封装axios请求方法风格(便于开发使用)
const axiosApi = {
    get:<T>(url,data,config)=> api.get<any,responseType<T>>(url,{...config,params:data}),
    post:<T>(url,data,config)=> api.post<any,responseType<T>>(url,data,config),
    put:<T>(url,data,config)=> api.put<any,responseType<T>>(url,data,config),
    delete:<T>(url,data,config)=> api.delete<any,responseType<T>>(url,{...config,data}),
}
export default axiosApi; //对外暴露接口

在src/api下新建user文件夹，用于管理用户相关的接口,并新建index.ts文件，用于导出接口。 ​

typescript
import axiosApi from "../index.ts"
import type {DictListType, Page, User} from "../type";
import {reactive, ref} from "vue";
//查询用户列表
export const getUserListApi = (params: {currentPage: number, pageSize: number}) => {
    return axiosApi.get<Page<User>>(`/userinfo/select/${params.currentPage}/${params.pageSize}`)
}
//新增用户
export const addUserApi = (user:User) => {
    return axiosApi.post<null>('/userinfo/insert', user)
}
//修改用户
export const updateUserApi = (user:User) => {
    return axiosApi.put<null>(`/userinfo/update`, user)
}
//删除用户
export const deleteUserApi = (userId:string) => {
    return axiosApi.delete<null>(`/userinfo/delete/${userId}`)
}
//获取字典的接口
export const getGenderDictApi = (...word:string[]) => {
    const dictMap:Record<string, DictListType[]> = {
    }
    word.forEach(key => {
        dictMap[key] = ref<DictListType[]>([])
        //分批请求字典数据
        axiosApi.get<DictListType[]>(`/select_dictionary/${key}`).then(res => {
            dictMap[key] = dictMap[key].value = res.data
        })
    })
    return  dictMap
}

  emit("submit", props.formData);

6.在src/page/user下完善index.vue文件 ​

vue
<template>
  <el-card v-loading="pageData.loading">
    <!-- 工具栏 -->
    <el-row align="middle" class="toolbar">
      <el-col :span="24">
        <!-- 新增用户按钮 -->
        <el-button type="primary" @click="handleAddOrEdit(null)">新增用户</el-button>
      </el-col>
    </el-row>

    <!-- 表格 -->
    <el-table :data="pageData.tableData" height="780px" border>
      <!-- ID列 -->
      <el-table-column prop="id" label="ID"></el-table-column>
      <!-- 用户名列 -->
      <el-table-column prop="userName" label="用户名"></el-table-column>
      <!-- 年龄列 -->
      <el-table-column prop="age" label="年龄"></el-table-column>
      <!-- 性别列 -->
      <el-table-column prop="gender" label="性别"></el-table-column>
      <!-- 邮箱列 -->
      <el-table-column prop="email" label="邮箱"></el-table-column>
      <!-- 操作列 -->
      <el-table-column label="操作" width="200px">
        <template #default="scope">
          <!-- 编辑按钮 -->
          <el-button link type="primary" @click="handleAddOrEdit(scope.row)">编辑</el-button>
          <!-- 删除按钮 -->
          <el-button link type="primary" @click="handleDelete(scope.row)">删除</el-button>
        </template>
      </el-table-column>
    </el-table>

    <el-divider />

    <!-- 分页组件 -->
    <el-pagination
        v-model:current-page="pageData.currentPage"
        v-model:page-size="pageData.pageSize"
        @current-change="getUsers"
        @size-change="() => { pageData.currentPage = 1; getUsers(); }"
        :page-sizes="pageData.pageSizes"
        :total="pageData.total"
        background
        layout="prev, pager, next,->,sizes,total,"
    ></el-pagination>

    <!-- 用户表单对话框 -->
    <formDialog
        v-model="pageData.dialogVisible"
        :form-data="formData"
        :post-code-list="post"
        @submit="handleSubmit"
    />
  </el-card>
</template>

<script lang="ts" setup>
import { onMounted, reactive } from "vue";
import formDialog from "./from.vue";
import type { User } from "../../api/type";
import { getUserListApi, getGenderDictApi, updateUserApi, addUserApi, deleteUserApi } from "../../api/user";
import { ElMessage } from "element-plus";

// 从字典 API 获取岗位列表
const { post } = getGenderDictApi("post");

// 初始化表单数据的方法
const initFormData = () => ({
  id: "",
  userName: "",
  age: 0,
  gender: "1", // 默认性别为男
  email: "",
  password: "",
  address: "",
  postCode: "",
});

// 表单数据（用于新增和编辑）
const formData = reactive(initFormData());

// 页面数据（用于显示表格数据）
const pageData = reactive({
  tableData: [], // 存放表格数据
  dialogVisible: false, // 对话框是否可见
  currentPage: 1, // 当前页码
  pageSize: 3, // 每页显示条数
  pageSizes: [3, 5, 7, 10, 20], // 每页显示条数选项
  total: 0, // 数据总条数
  loading: false, // 页面加载状态
});

// 获取用户列表的方法
const getUsers = async () => {
  try {
    pageData.loading = true; // 开始加载
    const result = await getUserListApi(pageData); // 调用接口获取数据
    pageData.tableData = result.data.list; // 更新表格数据
    pageData.total = result.data.total; // 更新总条数
  } catch (error) {
    console.error(error); // 打印错误日志
  } finally {
    pageData.loading = false; // 加载完成
  }
};

// 新增或编辑用户，打开表单对话框
const handleAddOrEdit = (user?: User) => {
  pageData.dialogVisible = true; // 显示对话框
  if (user) {
    // 如果传入用户数据，填充表单
    Object.assign(formData, user);
  } else {
    // 否则清空表单数据
    Object.assign(formData, initFormData());
  }
};

// 提交表单（新增或编辑）
const handleSubmit = async (user: User) => {
  try {
    let result;
    if (user.id) {
      // 编辑用户
      result = await updateUserApi(user);
    } else {
      // 新增用户
      result = await addUserApi(formData);
    }
    ElMessage.success(result.message); // 显示成功消息
    await getUsers(); // 重新加载用户列表
  } catch (error) {
    ElMessage.error(error.message); // 显示错误消息
  } finally {
    pageData.dialogVisible = false; // 关闭对话框
  }
};

// 删除用户
const handleDelete = async (user: User) => {
  try {
    const result = await deleteUserApi(user.id); // 调用删除接口
    ElMessage.success(result.message); // 显示成功消息
    await getUsers(); // 重新加载用户列表
  } catch (error) {
    ElMessage.error(error.message); // 显示错误消息
  }
};

// 页面加载时初始化用户列表
onMounted(() => {
  getUsers();
});
</script>

  • 如果你完成了上述步骤,那么恭喜你，你已经完成了用户管理的基本功能
  • 目前有一个问题,页面中涉及字典交互的地方,需要根据后端的接口返回数据做映射,要是每个地方都要写一遍代码,那岂不是太麻烦了?
  • 我们不妨思考一下,是否可以通过封装一个组件,来自动解决字典的映射问题?
  • 其实本质要干的事情不多,接收字典集合,根据选中的key,返回对应的label值即可.

首先在src/components下新建DictSelect.vue文件 ​

vue
<template>
  <el-tag type="primary" effect="dark" >{{label ? label : '无'}}</el-tag>
</template>
<script lang="ts" setup>
  import { computed} from "vue";
  import  {DictListType} from "../api/type";
  //使用element-plus的Tag类型便于类型提示
  import  {TagProps } from "element-plus";
  type DictSelectProps = {
    selected: string,
    dictList: DictListType[]
  } & Partial<TagProps>
  //设置可以接收的参数
  const props = withDefaults(defineProps<
      DictSelectProps>(), { //进行类型联合
    selected: '',
    dictList: [],
  })
  //根据value和dictList计算label
  const label = computed(() => props.selected && props.dictList.find(item => item.value === props.selected)?.label || '')
</script>
