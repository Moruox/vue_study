说明

在前面的章节中,已经对Vue的使用上有了一些基本了解,其中一个痛点，组件的开发工作,需要大量的样式,逻辑的编写,为了解决这个问题,阿里开源了Element-Plus组件库,它提供了一套常用功能的高质量组件,可以帮助我们快速开发出高质量的web应用, 本章节将介绍Element-Plus的使用方法,包括安装,组件的使用,以及一些常用组件的使用方法.

安装 ✳️ ​

bash
# 安装依赖
pnpm install element-plus
# 安装导入工具
pnpm install -D unplugin-vue-components unplugin-auto-import

基础布局 ⭐️ ​

在element-plus中，提供了一套完整的布局组件，可以帮助我们快速搭建页面的基本结构分别是常用的有el-row,el-col。

使用方法 ​

  • el-row: 行布局，会将当前宽度等切为24份，可以包含el-col组件。
  • el-col: 列布局，可以设置span属性，表示占据几份宽度，offset属性，表示左侧偏移几份宽度。
  • el-row和el-col组件可以嵌套使用，可以实现复杂的布局。
  • 当el-clo组件的的总宽度超过24份时，会自动换行。
  • 具体配置属性请参考官方文档

表格组件 ⭐️ ​

在element-plus中，提供了一套完整的表格组件，可以帮助我们快速实现各种类型的表格，由el-table与el-table-column组合使用。

使用方法 ​

  • el-table: 表格组件，可以包含el-table-column组件,组件可以设置data属性，表示表格的数据，
  • el-table-column: 表格列组件，可以设置prop属性，表示对应列的字段名，label属性，表示列的标题。
  • 具体配置属性请参考官方文档

分页器组件 ⭐️ ​

在element-plus中，提供了一套完整的分页器组件，可以帮助我们快速实现分页功能，由el-pagination组件组合使用。

使用方法 ​

  • el-pagination: 分页器组件，可以设置layout属性，表示分页器的布局，current-page属性，表示当前页，page-size属性，表示每页显示的条目数，total属性，表示总条目数。
  • 具体配置属性请参考官方文档

表单组件 ⭐️ ​

在element-plus中，提供了一套完整的表单组件，可以帮助我们快速实现各种类型的表单，由el-form与el-form-item及周边交互组件组合使用如el-input,el-select,el-date-picker,el-radio,el-checkbox等。

使用方法 ​

  • el-form: 表单组件，可以包含el-form-item组件,组件可以设置model属性，表示表单的数据对象，
  • el-form-item: 表单项组件，可以设置label属性，表示表单项的标题，prop属性，表示对应字段名。
  • 除了上述功能el-form组件还提供了rules属性，可以设置表单验证规则。
  • 具体配置属性请参考官方文档

对话框组件 ​

在element-plus中，提供了一套完整的对话框组件，可以帮助我们快速实现各种类型的对话框，由el-dialog组件组合使用。

使用方法 ​

  • el-dialog: 对话框组件，可以设置title属性，表示对话框的标题，v-model属性，表示对话框是否可见，width属性，表示对话框的宽度，before-close属性，表示对话框关闭前的回调函数。
  • 具体配置属性请参考官方文档
