区域划分 ​

说明

.vue文件由三个区域组成：模板<template>、脚本<script>、样式<style>。

  • 模板：.vue文件中的模板区域是HTML代码，它是.vue文件的核心，负责渲染页面的结构。
  • 脚本：.vue文件中的脚本区域是JavaScript代码，它是.vue文件的灵魂，负责处理数据和逻辑。
  • 样式：.vue文件中的样式区域是CSS代码，它是.vue文件的皮肤，负责美化页面的外观。
vue
<template>
</template>
<script>
</script lang="ts" setup>
<style>
</style>

说明

  • 脚本区域自带模块化支持,可以把这里理解为原生JavaScript代码，可以编写各种逻辑代码，比如数据处理、事件处理、方法调用等。
  • lang属性可以指定脚本语言，比如lang="ts"表示使用TypeScript语言。
  • setup属性是setup的语法糖，可以帮助我们更方便地使用Vue的组合式 API。

名词解释组合式 API

  • 组合式 API是Vue 3.0的语法风格，它提供了一种更高级的组件开发方式，使得代码更简洁。如同原生JavaScript一样，配合模块化思想，让代码便于维护和提取。 在Vue2版本中,则需要维护一个配置对象的形式来完成开发,并且还要关心this的指向问题。
  • Vu2风格
vue
<script>
export default {
 //使用data来到定义响应式数据
  data() {
    return {
      count: 0
    };
  },
 //使用methods来定义方法
  methods: {
    increment() {
      this.count++;
    }
  }
};
</script>

模版区域 ✳️ ​

vue
<template>
  <div>
    <h1>{{ userInfo.username }}</h1>
    <p>{{ userInfo.age }}</p>
    <p>{{ userInfo.address }}</p>
    <p>gender: {{ userInfo.gender === '1' ? '男' : '女' }}</p>
  </div>
</template>

  • 模板语法是.vue文件的核心，模板语法为HTML语法加上Vue的一些扩展语法组成(如指令、插槽等),可以帮助我们更方便地渲染页面。
  • 为了减少学习成本，本章只介绍模板语法的一些基础知识,指令和插槽的使用后续章节会详细介绍。
  • 模板区域内vue为我们提供了访问脚本区域数据的能力，需要使用{{}}语法来关联数据，{{}} 内部支持为表达式，所以我们可以直接使用JavaScript的语法来处理数据。如：{{ userInfo.username }},表示为使用<script lang="ts" setup>区域的userInfo对象中的username属性值渲染。
  • 模版中还允许在标签属性中读取脚本区域的数据，但是要使用 :[属性名]="变量名" 的语法，如：<input :value="userInfo.username">
  • 单使用:=变量名这种语法时，如果变量是对象，vue会将对象展开,以key='value'的形式赋值给标签的属性。

说明

如果在模版中，我们想要监听元素与用户交互行为时，只需要将原生语法on[事件名]替换成@[事件名]即可。比如：<button @click="(event) => {}">+</button>,当用户点击按钮时，会触发绑定的函数。

样式区域 ✳️ ​

vue
<style lang="css" scoped>
  h1 {
    color: blue;
  }
</style>

说明

  • 样式区域是书写CSS样式的地方，它可以帮助我们美化组件的样式。
  • lang属性可以指定样式语言，比如lang="css"表示使用CSS语言。
  • scoped属性可以让样式只作用于当前组件，不会影响全局样式,实现组件的样式隔离。
  • 与模版的配合使用，可以实现更复杂的样式效果。
  • 动态class语法
vue
<!-- 动态class[提到的布尔值]可以替换成脚本区域的变量与业务结合 -->
<template>
  <!--通过变量控制class样式生效情况-->
  <div class="box" :class="true ? 'blue-box' : 'red-box'"></div>
  <!--通过表达式控制class样式生效情况-->
  <div class="box" :class="{
      'blue-box': false,
       'red-box': true
  }"></div>
</template>
<style lang="css" scoped>
 .box {
    width: 100px;
    height: 100px;
  }
  .blue-box {
    background-color: blue;
  }
  .red-box {
    background-color: red;
  }
</style>

说明

  • 在样式区域中，也可以使用v-bind()的语法来访问脚本区域的数据。
vue
<template>
  <div class="box"></div>
</template>
<script setup>
import { ref } from 'vue'
const width = ref("200px")
const height = ref("300px")
</script>
<style>
.box {
  width: v-bind(width);
  height: v-bind(height);
  background-color: aqua;
}
</style>
