说明

  • 在vue中，每一个组件在不同的阶段会触发特定的函数，这些函数被称为生命周期函数。
  • 可以把组件理解为一个类从初始化->实例化完成->成员调整->被回收销毁的过程。
  • 开发中常用的生命周期函数有:onMounted、onUnMounted。
  • 完整的生命周期函数列表请参考官方文档。

onMounted ✳️ ​

  • onMounted函数在组件被解析完毕,渲染到页面之后执行,也是最稳定的生命周期函数。
  • 该函数可以用来处理一些异步请求、DOM操作、组件初始化等。
  • 该函数只会被调用一次，所以可以在该函数中进行一些初始化操作。
vue
<template>
  <div>
    <h1>{{ state.title }}</h1>
  </div>
</template>
<script lang="ts" setup>
  import { onMounted,reactive } from 'vue'
  const state = reactive({
    title: 'Hello world'
  })
  //模拟异步请求
  const fetchData =  () => {
      setTimeout(() => {
        Object.assign(state, { title: 'Hello Vue3.0' })
      },1000)
  }
  onMounted(() => {
    fetchData()
  })
</script>

onUnmounted ✳️ ​

  • onUnmounted函数在组件被销毁之前执行，可以用来清理一些定时器、取消请求、清理事件监听等。
  • 该函数只会被调用一次
vue
<template>
  <div>
    <h1>{{ state.count }}</h1>
  </div>
</template>
<script lang="ts" setup>
  import { onMounted,onUnmounted,reactive } from 'vue'
  let timer = null
  const state = reactive({
    count: 1
  })
  //定时更新数据源
  const fetchData =  () => {
    timer = setInterval(() => {
        state.count++
      },1000)
  }
  onMounted(() => {
    fetchData()
  })
  onUnmounted(() => {
    //清除定时器
    timer && clearInterval(timer)
  })
</script>
