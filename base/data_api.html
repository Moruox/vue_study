前面的篇章,有说到Vue当监听到数据源变化时,会自动更新模版中使用数据的视图, 其实在Vue中,数据源的要求必须是通过Vue内置函数来实现的数据源才会被Vue监听并自动更新视图,js内置的对象是做不到这一点的 本篇将介绍如何使用Vue内置的方法来实现响应式数据。 在此之前,我们先来看2组样例代码。

reactive ✳️ ​

reactive方法参数是一个对象，返回一个响应式对象，该对象中的所有属性都将变成响应式数据,后续对该对象的属性的修改将触发视图更新。

vue
<template>
  <!--模版访问state对象-->
  <div>{{ state.name }}</div>
  <div>{{ state.age }}</div>
</template>
<script lang="ts" setup>
  import {reactive} from "vue";
  const state = reactive({
    name: 'John',
    age: 20,
  })
  //当前区域访问state对象
  console.log(state.name)
</script>

ref ✳️ ​

ref方法参数是一个初始值，返回一个响应式对象，该对象中的所有属性都将变成响应式数据,后续对该对象的属性的修改将触发视图更新。 注意: ref方法返回的对象在<script setup>中使用时,需要使用要使用.value来获取其值,否则会报错,模版中使用时不需要.value。

vue
<template>
    <!--模版访问state对象，不需要使用.value-->
  <div>{{ state.name }}</div>
  <div>{{ state.age }}</div>
</template>
<script lang="ts" setup>
  import {ref} from "vue";
  const state = ref({
    name: 'John',
    age: 20,
  })
  //当前区域访问state对象，需要使用.value
  Object.assign(state.value, {name: 'Mike',age:18})
</script>

computed ✳️ ​

computed方法参数是一个函数，返回一个响应式对象，该对象会根据函数的返回值自动更新。

vue
<template>
  <!--模版访问state对象-->
  <div>{{ state.name }}</div>
  <div>{{ state.age }}</div>
  <div>{{ fullName }}</div>
</template>
<script lang="ts" setup>
  import {reactive, computed} from "vue";
  const state = reactive({
    name: 'John',
    lastName: 'Doe',
    age: 20,
  })
  //计算属性(如果state.name或state.lastName发生变化，fullName也会自动更新)
  const fullName = computed(() => `${state.name} ${state.lastName}`)
  //当前区域访问state对象
  console.log(fullName.value)
</script>

watch ✳️ ​

当响应式数据发生变化时,vue会自动更新视图,但有时我们需要在数据变化时执行一些额外的操作,watch方法可以帮助我们实现这个功能。
watch方法有3个参数:

  • source：要观察的响应式数据对象
  • cb：回调函数，在数据变化时执行
  • options：选项对象，
  • deep选项，如果为true，则会深度观察对象中的所有属性。
  • immediate选项：是否立即执行一次回调函数，默认为false。
vue
<template>
  <!--模版访问state对象-->
  <div>{{ state.name }}</div>
  <div>{{ state.age }}</div>
  <div>{{ fullName }}</div>
</template>
<script lang="ts" setup>
  import {reactive, computed, watch} from "vue";
  const state = reactive({
    name: 'John',
    lastName: 'Doe',
    age: 20,
  })    
  //计算属性(如果state.name或state.lastName发生变化，fullName也会自动更新)
  const fullName = computed(() => `${state.name} ${state.lastName}`)
  //监听state.name变化
  watch(
    () => state.name,
    (newVal, oldVal) => {
      console.log(`state.name changed from ${oldVal} to ${newVal}`)
    },
    {immediate: true}
  )
  //监听state.age变化
  watch(
    () => state.age,
    (newVal, oldVal) => {
      console.log(`state.age changed from ${oldVal} to ${newVal}`)
    },
    {immediate: true}
  )
  //当前区域访问state对象
  console.log(fullName.value)
</script>

watchEffect ✳️ ​

watchEffect方法与watch方法类似，但它是一种更简洁的写法，它只接收一个回调函数，当回调函数中涉及的响应式数据发生变化时，自动重新执行回调函数。

vue
<template>
  <!--模版访问state对象-->
  <div>{{ state.name }}</div>
  <div>{{ state.age }}</div>
  <div>{{ fullName }}</div>
</template>
<script lang="ts" setup>
  import {reactive, computed, watchEffect} from "vue";
  const state = reactive({
    name: 'John',
    lastName: 'Doe',
    age: 20,
  })    
  //计算属性(如果state.name或state.lastName发生变化，fullName也会自动更新)
  const fullName = computed(() => `${state.name} ${state.lastName}`)
  //监听state.age变化
  watchEffect(() => {
    if(state.age > 20){
        console.log(`state.age changed to ${state.age}`)
    }else{
        console.log(`state.age changed to ${state.age}, but it's too young`)
    }
  })
  //当前区域访问state对象
  console.log(fullName.value)
</script>
