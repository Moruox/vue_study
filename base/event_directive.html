说明

  • 指令是Vue中重要的概念，它可以用来绑定DOM元素的行为、实现数据双向绑定、监听事件、快速生成元素等功能。
  • 他们的存在本质上是为了让模版与数据之间的交换更加灵活、方便。

指令实现元素的隐藏与显示(v-if 和 v-show) ​

v-if ​

  • v-if指令可以用来控制元素的显示与隐藏，当表达式的值为true时，元素才会被渲染，否则元素会被销毁。
  • v-if也可以与v-else指令配合使用，实现条件渲染，即当表达式的值为false时，才渲染v-else中的内容。
  • 使用时，v-if指令必须放在元素的开始标签中，v-else指令必须放在v-if指令的结束标签中。

v-show ​

  • v-show指令的作用与v-if指令类似，也是用来控制元素的显示与隐藏，不同的是v-show指令始终会渲染元素，只是通过display:none来控制元素的显示与隐藏。
vue
<template>
  <div v-if="true">v-if</div>
  <div v-else>v-else</div>
  <div v-show="false">v-show</div>
</template>

指令实现数据的双向绑定(v-model) ​

v-model指令可以实现数据的双向绑定，即input元素的值与data绑定，data的值与input元素的值绑定。

vue
<template>
  <input type="text" v-model="message">
  <p>{{ message }}</p>
</template>

指令实现事件的监听(v-on) ​

v-on指令可以用来监听事件，并执行相应的回调函数,语法为v-on:事件名="回调函数"。当然，也可以简写为@事件名="回调函数"。

vue
<template>
  <button v-on:click="handleClick">点击</button>  
    <!--需要传入参数时，可以这样写-->
  <button @click="handleClick(true)">点击</button>
  <button @click="handleClick">点击</button>
</template>
<script lang="ts" setup>
const handleClick = (flag?: boolean) => {
  console.log(`按钮被点击了当前的flag值是：${flag}`)
}
</script>

指令实快速现循环生成元素(v-for) ​

  • v-for指令可以用来快速生成元素，语法为v-for="(item, index) in items"。
  • items支持的类型为数组、对象、数字,数组根据长度影响生成的元素个数，对象为键值对，数字则范围内生成元素个数，
  • item为数组或对象中的每一项，index为当前项的索引。
vue
<template>
  <ul>
    <!--你将会看到apple、banana、orange 3个元素被渲染-->
    <li v-for="(item, index) in items" :key="index">{{ item }}</li>
  </ul>
</template>
<script lang="ts" setup>
const items = ['apple', 'banana', 'orange'] 
</script>

注意

key这个属性，是Vue用来帮助它更快地跟踪每个节点的身份的特殊属性，它的值必须是每项都独一无二的。 非不要请勿使用index作为key的值，因为它会导致性能下降，并且在列表更新时会出现问题。

说明

  • 以上指令为Vue中的常用内置指令，还有很多其他的指令，如v-once、v-pre、v-slot或自定义指令等，可以参考官方文档进行了解Vue指令官方文档。
