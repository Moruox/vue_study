快速上手 ✳️ ​

  • pigx本身已经提供了丰富了组件和详细的使用温度,请参考官方文档。
  • 现在我们将取一个简单的内置模块来为大家快速上手。
  • 模块所在目录src/views/admin/system/role/index.vue
vue
<!--src/views/admin/system/role/index.vue-->
<template>
  <div class="layout-padding">
    <!-- 搜索栏 -->
    <div class="layout-padding-auto layout-padding-view">
      <el-row shadow="hover" v-show="showSearch" class="ml10">
        <!-- 查询表单 -->
        <el-form :model="state.queryForm" ref="queryRef" :inline="true" @keyup.enter="getDataList">
          <el-form-item :label="$t('sysrole.roleName')" prop="roleName">
            <el-input :placeholder="$t('sysrole.inputRoleNameTip')" v-model="state.queryForm.roleName" />
          </el-form-item>
          <el-form-item>
            <!-- 查询按钮 -->
            <el-button icon="search" type="primary" @click="getDataList">
              {{ $t('common.queryBtn') }}
            </el-button>
            <!-- 重置按钮 -->
            <el-button icon="Refresh" @click="resetQuery">{{ $t('common.resetBtn') }}</el-button>
          </el-form-item>
        </el-form>
      </el-row>

      <!-- 操作栏 -->
      <el-row>
        <div class="mb8" style="width: 100%">
          <!-- 添加角色按钮 -->
          <el-button icon="folder-add" type="primary" class="ml10" @click="roleDialogRef.openDialog()" v-auth="'sys_role_add'">
            {{ $t('common.addBtn') }}
          </el-button>
          <!-- 导入角色按钮 -->
          <el-button plain icon="upload-filled" type="primary" class="ml10" @click="excelUploadRef.show()" v-auth="'sys_user_add'">
            {{ $t('common.importBtn') }}
          </el-button>
          <!-- 删除按钮（多选删除） -->
          <el-button plain :disabled="multiple" icon="Delete" type="primary" class="ml10" v-auth="'sys_user_del'" @click="handleDelete(selectObjs)">
            {{ $t('common.delBtn') }}
          </el-button>
          <!-- 右侧工具栏 -->
          <right-toolbar
              v-model:showSearch="showSearch"
              :export="'sys_role_export'"
              @exportExcel="exportExcel"
              class="ml10"
              style="float: right; margin-right: 20px"
              @queryTable="getDataList"
          ></right-toolbar>
        </div>
      </el-row>

      <!-- 角色表格 -->
      <el-table
          :data="state.dataList"
          v-loading="state.loading"
          style="width: 100%"
          row-key="roleId"
          @selection-change="handleSelectionChange"
          border
          :cell-style="tableStyle.cellStyle"
          :header-cell-style="tableStyle.headerCellStyle"
      >
        <!-- 多选列 -->
        <el-table-column type="selection" :selectable="handleSelectable" width="50" align="center" />
        <!-- 序号列 -->
        <el-table-column type="index" :label="$t('sysrole.index')" width="80" />
        <!-- 角色名称 -->
        <el-table-column prop="roleName" :label="$t('sysrole.roleName')" show-overflow-tooltip></el-table-column>
        <!-- 角色编码 -->
        <el-table-column prop="roleCode" :label="$t('sysrole.roleCode')" show-overflow-tooltip></el-table-column>
        <!-- 角色描述 -->
        <el-table-column prop="roleDesc" :label="$t('sysrole.roleDesc')" show-overflow-tooltip></el-table-column>
        <!-- 数据权限 -->
        <el-table-column prop="data_authority" :label="$t('sysrole.data_authority')" show-overflow-tooltip>
          <template #default="scope">
            <dict-tag :options="dictType" :value="scope.row.dsType"></dict-tag>
          </template>
        </el-table-column>
        <!-- 创建时间 -->
        <el-table-column prop="createTime" :label="$t('sysrole.createTime')" show-overflow-tooltip></el-table-column>
        <!-- 操作列 -->
        <el-table-column :label="$t('common.action')" width="250">
          <template #default="scope">
            <!-- 编辑角色按钮 -->
            <el-button text type="primary" icon="edit-pen" v-auth="'sys_role_edit'" @click="roleDialogRef.openDialog(scope.row.roleId)">{{
              $t('common.editBtn')
              }}</el-button>
            <!-- 权限配置按钮 -->
            <el-button text type="primary" icon="turn-off" v-auth="'sys_role_perm'" @click="permessionRef.openDialog(scope.row)">{{
              $t('sysrole.permissionTip')
              }}</el-button>
            <!-- 删除角色按钮 -->
            <el-tooltip :content="$t('sysrole.deleteDisabledTip')" :disabled="scope.row.roleId !== '1'" placement="top">
              <span style="margin-left: 12px">
                <el-button
                    text
                    type="primary"
                    icon="delete"
                    :disabled="scope.row.roleId === '1'"
                    v-auth="'sys_role_del'"
                    @click="handleDelete([scope.row.roleId])"
                >{{ $t('common.delBtn') }}</el-button>
              </span>
            </el-tooltip>
          </template>
        </el-table-column>
      </el-table>
      <!-- 分页组件 -->
      <pagination @size-change="sizeChangeHandle" @current-change="currentChangeHandle" v-bind="state.pagination" />
    </div>

    <!-- 角色编辑/新增对话框 -->
    <role-dialog ref="roleDialogRef" @refresh="getDataList()" />
    <!-- 角色导入对话框 -->
    <upload-excel
        ref="excelUploadRef"
        :title="$t('sysuser.importUserTip')"
        url="/admin/role/import"
        temp-url="/admin/sys-file/local/file/role.xlsx"
        @refreshDataList="getDataList"
    />
    <!-- 权限配置对话框 -->
    <permession ref="permessionRef" />
  </div>
</template>

<script setup lang="ts" name="systemRole">
  /*
    主要功能：系统角色管理页面
    功能模块：
    1. 搜索功能：根据角色名称进行查询。
    2. 操作功能：
       - 添加角色
       - 导入角色
       - 多选删除角色
    3. 表格展示：显示角色信息，包括名称、编码、描述、权限等。
    4. 对话框：支持角色新增/编辑、权限配置以及导入操作。
    5. 导出功能：支持导出角色数据。
  */

  // 引入必要的钩子与模块
  import { BasicTableProps, useTable } from '/@/hooks/table';
  import { pageList, delObj } from '/@/api/admin/role';
  import { useMessage, useMessageBox } from '/@/hooks/message';
  import { useI18n } from 'vue-i18n';
  import { defineAsyncComponent, reactive, ref } from 'vue';

  // 异步加载子组件
  const RoleDialog = defineAsyncComponent(() => import('./form.vue')); // 角色编辑/新增表单
  const Permession = defineAsyncComponent(() => import('./permession.vue')); // 权限配置组件
  const { t } = useI18n(); // 国际化翻译

  // 定义全局引用变量
  const roleDialogRef = ref(); // 角色对话框引用
  const permessionRef = ref(); // 权限对话框引用
  const excelUploadRef = ref(); // Excel上传组件引用
  const queryRef = ref(); // 查询表单引用
  const showSearch = ref(true); // 是否显示查询栏
  const selectObjs = ref([]) as any; // 选中的角色ID集合
  const multiple = ref(true); // 是否允许多选

  // 状态变量
  const state: BasicTableProps = reactive<BasicTableProps>({
    queryForm: { roleName: '' }, // 查询表单初始值
    pageList: pageList, // 表格数据加载方法
    descs: ['create_time'], // 默认排序
  });

  // 数据权限类型字典
  const dictType = ref([
    { label: '全部', value: '0' },
    { label: '自定义', value: '1' },
    { label: '本级及子级', value: '2' },
    { label: '本级', value: '3' },
    { label: '本人', value: '4' },
  ]);

  // 表格操作钩子
  const { getDataList, currentChangeHandle, sizeChangeHandle, downBlobFile, tableStyle } = useTable(state);

  // 清空查询条件并刷新列表
  const resetQuery = () => {
    queryRef.value.resetFields();
    getDataList();
  };

  // 导出角色数据为 Excel
  const exportExcel = () => {
    downBlobFile('/admin/role/export', Object.assign(state.queryForm, { ids: selectObjs }), 'role.xlsx');
  };

  // 判断是否允许多选
  const handleSelectable = (row: any) => row.roleId !== '1';

  // 多选行的事件处理
  const handleSelectionChange = (objs: { roleId: string }[]) => {
    selectObjs.value = objs.map(({ roleId }) => roleId);
    multiple.value = !objs.length;
  };

  // 删除选中角色
  const handleDelete = async (ids: string[]) => {
    try {
      await useMessageBox().confirm(t('common.delConfirmText')); // 确认弹框
      await delObj(ids); // 调用删除接口
      getDataList(); // 刷新列表
      useMessage().success(t('common.delSuccessText')); // 删除成功提示
    } catch (err: any) {
      useMessage().error(err.msg); // 删除失败提示
    }
  };
</script>

说明

  • 🤣相信你一定是一路滚轮来到这里的吧，是不是看起来一坨，头皮发麻(不急结合下面的内容,再看效果更佳)~~~
  • 不要慌,接下来我们将详细拆解一下这个模块。
  • 其实从上面的代码中,我们可以看出,这个模块的大部分组件都来自element-plus库,想必通过前面的学习,大家已经对element-plus有了一定地了解(你不会没有看过前面的章节吧😂)。

t函数 ✳️ ​

  • 🙋是不是看到了很多$t('xxx')或t('xxx')这样的语法,其实这是国际化的配置函数。
  • t函数是vue-i18n提供的全局函数,它可以帮助我们在模板中实现多语言切换。
  • 他的工作原理,在全局有个语言标识来标记当前的语言,然后根据标识来获取对应的语言包,然后根据key来获取对应的翻译内容。
  • 语言包的位置在src/i18n/lang/目录下,每个语言包是一个json文件,文件名即为语言标识。
  • 如t('sysrole.roleName')会根据标识选择读取哪个文件,然后根据key获取对应的翻译内容。

useTable函数 ✳️ ​

  • useTable函数是src/hooks/table.ts文件中的一个函数,它主要是用来处理表格数据的。
  • 它接收一个BasicTableProps类型参数,它包含了表格的一些基本属性,包括queryForm、pageList、descs等。
  • 他会接收一个查询方法和查询条件,在页面onMounted函数中自动调用一次getDataList方法,来加载表格数据。
  • 它会返回一些状态变量和方法,包括getDataList、currentChangeHandle、sizeChangeHandle、downBlobFile、tableStyle等。
  • 相当于把一些常用的逻辑抽离,方便复用。
  • 我们可以直接使用useTable函数来处理表格数据,而不用再去写这些重复的代码。

dict-tag组件 ✳️ ​

  • dict-tag组件是src/components/DictTag.vue文件中的一个组件,它主要是做字典映射的。
  • 字典映射😆是不是好像在哪见过,没错,就是又是在之前的章节中提到的字典映射功能。
  • 它接收一个options属性,它是一个数组,数组的每个元素是一个对象,对象有label和value属性。
  • 它会根据value属性的值来匹配label属性,然后显示出来。
  • 功能用法和我们之前封装的dict-select组件完全一致,理论上来讲你已经是个字典映射高手了。

role-dialog组件 ✳️ ​

  • role-dialog组件是src/views/admin/system/role/form.vue文件中的一个组件,它主要是用来编辑/新增角色的。
  • 它接收一个ref属性,它是一个vue的ref对象,用来获取对话框的实例,可以调用openDialog方法来打开对话框。。
  • 它会根据ref属性的值来判断是新增还是编辑,然后显示不同的标题和按钮。
  • 它会接收一个refresh事件,当角色保存成功后,会触发refresh事件,刷新列表数据。
  • 功能用法和我们之前封装的form.vue组件及其相似,本质上变化不大。

upload-excel组件 ✳️ ​

  • upload-excel组件是src/views/admin/system/role/UploadExcel.vue文件中的一个组件,它主要是用来导入角色的。
  • 它接收一个ref属性,它是一个vue的ref对象,用来获取对话框的实例。
  • 它会接收一个title属性,用来设置弹框的标题。
  • 它会接收一个url属性,用来设置上传文件的接口地址。
  • 它会接收一个temp-url属性,用来设置临时文件上传的接口地址。
  • 它会接收一个@refreshDataList事件,当文件上传成功后,会触发refreshDataList事件,刷新列表数据。

permession组件 ✳️ ​

  • permession组件是src/views/admin/system/role/permession.vue文件中的一个组件,它主要主要作用是开启对话框进行角色的权限配置。
  • 它接收一个ref属性,它是一个vue的ref对象,用来获取对话框的实例,可以调用openDialog方法来打开对话框。

v-auth指令 ✳️ ​

  • v-auth指令是src/directives/auth.ts文件中的一个函数,它主要是用来控制页面元素的显示和隐藏。
  • 它接收一个value属性,它是一个字符串,用来标记当前用户的权限标识。
  • 它会根据value属性的值来判断当前用户是否有权限显示该元素,如果有权限,则显示,否则隐藏。
  • 而用户的权限标识,则是由后端返回给前端的/admin/user/info接口中permissions字段提供。

总结 ​

  • 是不是细化拆分后,觉得这个模块的结构清晰了许多.
  • 其实只要你认真完成了前面的章节,这一章的内容你都可以轻松应付,只是需要一点小小的耐心阅读理解。
